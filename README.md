# Banking System (C++)

A simple console-based banking system built as a beginner C++ project.

## 
- User registration and login
- Deposit and withdraw money
- Transfer between users
- Balance checking
- File-based data storage
## 
- C++
- Object-Oriented Programming
- File handling
- std::vector

## Notes
This is a beginner-level project created for learning purposes.  
Passwords are stored as plain text and security features are basic.
